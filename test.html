CSS

.p-0
Set padding to 0 for the element
.p-sm-3
Set padding of 3 for the element on small screens
.pl-2
Set left padding of 2 for the element
.pl-md
Set left padding to medium
.pr-lg
Set right padding to large
.mt-auto
s
Set auto margin-top for the element
.mb-sm-5
Set bottom margin of 5 for the element on all but small screens
.mrgn-tp-0
Set top margin to 0 for the element
.mrgn-bttm-0
Set bottom margin to 0 for the element
.mrgn-lft-0
Set left margin to 0 for the element
.mrgn-rght-0
Set right margin to 0 for the element
.mrgn-tp-sm
Set top margin to 5px for the element
.mrgn-lft-md
Set left margin to 15px for the element
.mrgn-rght-lg
Set right margin to 30px for the element

Appearance

No padding

Padding of 3 on small screens

Left padding of 2

Medium left padding (15px)

Large right padding (30px)

Auto margin-top

Bottom margin of 5 on all but small screens

Top margin of 0

Bottom margin of 0

Medium left margin (15px)

Large right margin (30px)

Code

// No padding
		<p class="p-0">...</p>
// Padding of 3 on small screens
		<p class="p-sm-3">...</p>
// Left padding of 2
		<p class="pl-2">...</p>
// Medium left padding
		<p class="pl-md">...</p>
// Large right padding
		<p class="pr-lg">...</p>
// Auto margin-top
		<p class="mt-auto">...</p>
// Bottom margin of 5 on small screens
		<p class="mb-sm-5">...</p>
// Top margin of 0
		<p class="mrgn-tp-0">...</p>
// Bottom margin of 0
		<p class="mrgn-bttm-0">...</p>
// Left margin of 15px
		<p class="mrgn-lft-md">...</p>
// Right margin of 30px
		<p class="mrgn-rght-lg">...</p>
